composer-rackspace
==================

Wrapper for the racksapce open cloud API. Installable as a code ingniter libary using CI config files.

```php

$config['AUTHURL'] = 'https://lon.identity.api.rackspacecloud.com/v2.0/';
$config['USERNAME'] = '';
$config['APIKEY'] = '';
$config['CONTAINER'] = '';

```
